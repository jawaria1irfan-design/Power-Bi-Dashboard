# Power-Bi-Dashboard
# 🏥 Healthcare Wait List Dashboard — Power BI

A Power BI dashboard analyzing hospital patient wait list data 
across multiple dimensions.

<img width="659" height="391" alt="image" src="https://github.com/user-attachments/assets/576851e1-2bb5-4fa5-acae-23aabe819d72" />




## 📊 Dashboard Overview

- **Latest Month Wait List:** 709K patients
- **PY Latest Month Wait List:** 640K patients
- Supports toggle between **Average** and **Median** views

## 🔍 Key Visuals

| Visual | Description |
|---|---|
| Donut Chart | Wait list split by Case Type (Outpatient, Day Case, Inpatient) |
| Grouped Bar Chart | Avg/Med wait list by Time Bands and Age Profile |
| Line Charts | Sum of Total by Archive Date and Case Type |
| KPI Cards | Wait times by Speciality (A&E, Cardiology, Breast Surgery, etc.) |

## 🛠️ Tools Used

- Microsoft Power BI Desktop
- Dataset: Hospital Patient Wait List Records

## 🎯 Filters Available

- Speciality Name
- Case Type
- Archive Date Range (Date Slicer)
